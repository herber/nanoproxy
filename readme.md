# Nanoproxy

> Simple reverse proxy

## Install

```
$ npm install -g nanoproxy
```

## Usage

```
$ nanoproxy

  Usage
    nanoproxy <from> <to>

  Options
    --logger, -l Specifies the logging format for morgan  (default common)

  Examples
    $ nanoproxy proxy 3000 80
    $ nanoproxy proxy http://example.com 80
```
